# Mode

Processing mode for the parser. `fast` favors speed, `accurate` (pro subscription only) favors quality and layout fidelity, and `balanced` offers a compromise between the two. Defaults to `balanced`.


## Values

| Name       | Value      |
| ---------- | ---------- |
| `FAST`     | fast       |
| `BALANCED` | balanced   |
| `ACCURATE` | accurate   |
